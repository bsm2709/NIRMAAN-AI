## üìã Key Points to Remember

### Elevator Pitch (30 seconds)
"Nirmaan AI is an intelligent construction progress monitoring system that uses AI/ML to analyze construction site images and predict project progress and delays. It's a full-stack web application with role-based access for citizens, officials, and administrators."

### Tech Stack
- **Backend**: Flask + SQLAlchemy + JWT + TensorFlow
- **Frontend**: React + Material-UI + Leaflet
- **Database**: SQLite (dev) / PostgreSQL (prod)
- **AI/ML**: CNN models + YOLOv8

### Core Features
1. AI-powered progress prediction
2. Role-based dashboards
3. Interactive maps
4. Progress tracking
5. Community engagement

### AI Models
1. **Stage Classification**: CNN classifies construction stage (0-5)
2. **Delay Prediction**: Hybrid model (image + tabular data)
3. **Object Detection**: YOLOv8 for building detection

### User Roles
- **Citizen**: View projects, track progress, submit comments
- **Official**: Manage projects, update progress, respond to comments
- **Admin**: Full system access, manage all projects/users

### Sample Credentials
- Admin: `admin` / `admin123`
- Official: `official1` / `official123`
- Citizen: `santosh` / `santosh123`
